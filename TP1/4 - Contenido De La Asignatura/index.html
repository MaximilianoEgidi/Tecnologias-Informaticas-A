Contenido de la asignatura

  • Presentación. Conceptos básicos de web y maquetación. Editor de código. Conceptos básicos de HTML. Estructura básica de un documento HTML

  • Estructura elemento HTML. Atributos comunes a etiquetas. Modelo de cajas. Introducción a CSS. Selectores.

  • HTML y CSS: Comentarios. Etiquetas de texto. Listas. Tablas. Contenidos embebidos: Imagen. Video. Audio.

  • Concepto Cliente/Servidor. Protocolo HTTP. Servidor HTTP. Registro de dominio. Publicación web.

  • HTML y CSS: Etiquetas DIV y SPAN. Otras etiquetas de agrupación. CSS: Propiedad Display. HTML: Etiquetas semánticas de sección.

  • Unidades en CSS: Absolutas, relativas y del viewport. CSS: Propiedad Position. CSS: Funciones.

  • HTML y CSS: Respondive. Etiqueta META.

  • Formularios: Estructura. Métodos GET y POST. Atributos.
