Estructura Basica De Un Documento HTML

!DOCTYPE

Informa al navegador qué versión de HTML se usó para escribir el documento.

HTML

Indica que el documento es un documento HTML

HEAD

Cabecera de la página. Contiene elementos que ayudarán a que la página funcione correctamente.

BODY

Cuerpo de la página. Contiene prácticamente todos los elementos visuales de la página
