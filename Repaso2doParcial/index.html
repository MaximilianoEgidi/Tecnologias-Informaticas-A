Metodo HTPP Significado de Restful Web Services
GET Se utiliza para operaciones de solo lectura. No generan ningun cambio en el servidor.
DELETE Elimina un recurso en especifico. Ejecutar esta operacion multiples veces no tiene ningun efecto.
POST Cambia la informacion de un recurso en el servidor. Puede o no regresar informacion.
PUT Almacena informacion de un recurso particular. Ejecutar esta operaciones multiples veces no tiene efecto, ya que se esta almacenando la misma informacion sobre el recurso.
Head Regresa solo el codigo de respuesta y cualquier cabecero HTTP asociado con la respuesta.
OPTIONS Representa las opciones disponibles para establecer la comunicacion en el proceso de peticion/resupuesta de una URL.
